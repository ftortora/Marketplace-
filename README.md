Le Pagine
Registrazione
Puoi compilare il form per provare la validazione in tempo reale, ma ricorda che i dati non vengono salvati da nessuna parte (non c'è un server dove memorizzarli).
Requisiti password per testare la validazione:

Minimo 8 caratteri
Almeno una maiuscola
Almeno un numero
Almeno un carattere speciale

Login
Per accedere all'app usa queste credenziali:
Username: admin
Password: Password123!
Semplice no? Essendo una demo senza backend, le credenziali sono già impostate nel codice.
Home - Il Catalogo
Qui trovi tutti i prodotti disponibili:

Cerca quello che ti serve usando la barra di ricerca
Guarda i dettagli di ogni prodotto
Aggiungi al carrello quello che ti piace

Admin - Gestisci i Prodotti
Questa è la tua area personale per:

Creare nuovi prodotti (nome, brand, descrizione, foto, prezzo)
Modificare quelli esistenti
Eliminare quelli che non ti servono più
Vedere l'anteprima delle immagini mentre le aggiungi

Carrello
Qui controlli tutto prima di "acquistare":

Vedi tutti i prodotti che hai scelto
Rimuovi quelli che non ti convincono più
Svuota tutto il carrello se cambi idea
Procedi al checkout (è simulato, tranquillo!)


Cosa puoi fare

Gestire i prodotti: creali, modificali, eliminali come preferisci
Il carrello si salva: anche se chiudi il browser, i tuoi prodotti restano lì
Cerca velocemente: digita nella barra di ricerca e trova subito quello che cerchi
Funziona ovunque: da computer, tablet o smartphone
Notifiche amichevoli: ogni azione ti mostra un messaggio per confermare che è andato tutto bene


Come funziona
Vai al Login → Esplora il Catalogo → Aggiungi al Carrello → Fai il Checkout
                    ↓
        Oppure vai in Admin e gestisci i tuoi prodotti

Menu di Navigazione

Home - torna al catalogo

Carrello - con il numerino dei prodotti aggiunti
Logout - per uscire

Tecnologie Usate

HTML5, CSS3, JavaScript
Bootstrap 5.3 per il layout
Font Awesome 6.4 per le icone
API Striveschool per i prodotti


Cose da Sapere

Il checkout è finto (nessun pagamento reale)
Il login è locale (non c'è un vero server)
Il carrello si salva nel browser
I prodotti arrivano da un'API esterna


Design

Colori: Blu e Verde moderni
Font: Inter (bello e leggibile)
Stile: Minimale e pulito
